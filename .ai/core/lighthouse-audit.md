# Lighthouse Governance

Target:
- Performance: 95+
- Accessibility: 95+
- Best Practices: 95+
- SEO: 95+

Mandatory:
- optimized LCP
- low CLS
- minimal blocking JS
- lazy hydration
- optimized fonts
- optimized images
- optimized script loading

Forbidden:
- render-blocking animation systems
- oversized JS bundles
- unoptimized media
- expensive layout calculations
- unnecessary hydration

Performance priorities:
1. LCP
2. INP
3. CLS
4. memory usage
5. bundle size
