# Accessibility Governance

Mandatory:
- semantic HTML
- keyboard navigation
- aria labels
- reduced motion support
- focus visibility
- accessible contrast

Never:
- rely purely on hover
- remove focus outlines blindly
- hide critical content visually only
- ignore reduced motion preferences

Motion systems must:
- support prefers-reduced-motion
- degrade gracefully

Typography:
- readable line lengths
- readable spacing
- accessible hierarchy
